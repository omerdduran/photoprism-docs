# Scans #
You can mark scanned photos as *scans* within the photo edit dialog.

We aim to automatically mark scans in the future.

## Set Scan Flags ##

 1. Open the [*photo edit dialog*](edit.md)
 2. Click :material-cog:
 3. Set or unset the scan flag
